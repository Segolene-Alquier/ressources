Nos mentors peuvent vous accompagner sur les sujets suivants :
* trouver son 1er emploi de développeuse
* préparer les tests techniques
* les compétences techniques de base pour travailer en entreprise (git, shell basics, tests, ...)
* pair-programmer sur un de vos projets
